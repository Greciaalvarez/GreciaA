Zarella Signal EA: Automated Forex System

Algorithmic Trading System developed by Grecia Zarella Alvarez Leyva

MQL5 (C++) MetaTrader 5 Algorithmic Trading Risk Management

1. The Problem: Manual Execution Constraints

This project was built from scratch to remove human emotion from trading and enforce strict risk management rules. By transitioning from Python analysis to an MQL5 Expert Advisor (EA), the system achieves 24/7 autonomous market monitoring, executing trades based on quantitative rules rather than discretionary decisions.

2. Core Engineering Features

Multi-Timeframe Analysis

The EA validates H4 trend bias before executing H1 trades, ensuring the system only enters in the direction of the higher-timeframe momentum.

Dynamic Risk Management

Features an automated Lot Size Calculator based on account equity and a structural ATR-based Stop Loss, ensuring consistent risk per trade.

3. System Architecture

The system leverages a "Pullback and Resumption" logic. Below is the snippet demonstrating the execution engine that manages risk-adjusted position sizing and trade modifications:

// ATR-Based Stop Loss & Position Sizing Logic double sl = structuralLow - (atr * 1.72); double lot = CalculateLotSize(MathAbs(close1 - sl)/_Point); double tp = close1 + (MathAbs(close1 - sl) * 2); OpenTradeBuy(lot, sl, tp);

4. Strategic Conclusion

Developing this EA taught me the difference between a "backtested strategy" and a "live system." Dealing with execution errors, liquidity filters, and unexpected market volatility while the system is running live has refined my ability to architect robust, production-ready code. This system is not just about signal generation; it is about algorithmic discipline.
